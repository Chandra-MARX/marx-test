Setting up marx accuracy and functional tests
=============================================

This small help file documents the ``marx-test`` module. The source code can be found on https://github.com/Chandra-MARX/marx-test . This is **not** the documention for the marx code and **not** even a list of marx test results (both can be found at http://space.mit.edu/cxc/marx/ ). Instead, this page documents the code used to run the suite of marx tests and shows how to add more test cases.
